# Tsubame

## Install Dependency

```sh
pip install cherrypy
```

## Config

Create `config.json`:

```json
{ "username": "admin"
, "password": "123456"
}
```

## Run

```sh
./main.py
```

Server would start on <http://127.0.0.1:8080>

Access log would be logged in `access.log`

Error log would be logged in `error.log`
